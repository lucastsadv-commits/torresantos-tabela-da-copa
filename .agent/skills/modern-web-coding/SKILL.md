# Modern Web Coding

Skill focada em excelência técnica e performance.

## Stack Padrão
- **Runtime**: React 18+ (Vite)
- **Styling**: Tailwind CSS (Utility-first)
- **Types**: TypeScript (Strict mode)

## Diretrizes Técnicas

1. **Responsividade**: Mobile-first é obrigatório.
2. **Componentização**: Mantenha lógica separada da UI na medida do possível.
3. **SEO**: Tags semânticas (`main`, `section`, `article`) e hierarquia de títulos correta.
4. **Otimização**: Lazy loading para imagens e componentes pesados.

## Fluxo de Desenvolvimento

1. Configurar infraestrutura base (Vite/Tailwind).
2. Criar sistema de design (tokens/variáveis).
3. Desenvolver componentes reutilizáveis.
4. Otimizar bundle e métricas de performance (Lighthouse).
