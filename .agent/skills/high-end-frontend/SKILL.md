# High-End Frontend Design

Cria interfaces frontend production-grade com alta qualidade de design. Ativa quando o usuário pede componentes web, páginas, aplicações, posters — websites, landing pages, dashboards, componentes React, layouts HTML/CSS, ou qualquer estilização de UI. Gera código criativo e refinado que evita a estética genérica de IA.

## Design Thinking (antes de qualquer código)
Entenda o contexto e defina uma direção estética clara antes de escrever uma linha:

- **Propósito** — Que problema essa interface resolve? Quem usa?
- **Tom** — Escolha um extremo: brutalmente minimalista, maximalista, retrô-futurista, orgânico/natural, luxo/refinado, lúdico, editorial/revista, brutalista/cru, art déco/geométrico, suave/pastel, industrial/utilitário.
- **Restrições** — Framework, performance, acessibilidade.
- **Diferenciação** — O que torna isso inesquecível? O que o usuário vai lembrar?

**CRÍTICO:** Escolha uma direção conceitual e execute com precisão. Maximalismo arrojado e minimalismo refinado funcionam — o que não funciona é falta de intenção.

## Diretrizes de Estética Frontend

### Tipografia
Escolha fontes belas, únicas e interessantes. **Proibido:** Arial, Inter, Roboto e fontes do sistema. Prefira escolhas distintas que elevem a estética — inesperadas, com caráter. Combine uma fonte de display marcante com uma fonte de corpo refinada.

### Cor e Tema
Comprometa-se com uma estética coesa. Use variáveis CSS para consistência. Cores dominantes com acentos fortes superam paletas tímidas e distribuídas de forma uniforme.

### Motion
Use animações para efeitos e micro-interações. Priorize soluções CSS puro para HTML. Use a biblioteca Motion para React quando disponível. Foque nos momentos de alto impacto: um carregamento de página bem orquestrado com reveals escalonados (animation-delay) gera mais encanto do que micro-interações espalhadas. Use scroll-triggering e hover states surpreendentes.

### Composição Espacial
Layouts inesperados. Assimetria. Sobreposição. Fluxo diagonal. Elementos que quebram o grid. Espaço negativo generoso OU densidade controlada.

### Fundos e Detalhes Visuais
Crie atmosfera e profundidade em vez de usar cores sólidas como padrão. Adicione efeitos e texturas contextuais que combinem com a estética geral. Aplique formas criativas como gradient meshes, texturas de noise, padrões geométricos, transparências em camadas, sombras dramáticas, bordas decorativas, cursores personalizados e overlays de grain.

## O que NUNCA fazer
- Famílias de fontes genéricas de IA: Inter, Roboto, Arial, fontes do sistema
- Esquemas de cor clichê (especialmente gradientes roxos em fundo branco)
- Layouts e padrões de componentes previsíveis
- Design cookie-cutter sem caráter específico ao contexto
- Nunca convergir para escolhas comuns (Space Grotesk, por exemplo) entre gerações diferentes

Cada design deve ser diferente. Varie entre temas claros e escuros, fontes diferentes, estéticas diferentes.

## Regra de Complexidade por Estética
Combine a complexidade da implementação com a visão estética:
- **Designs maximalistas** → código elaborado com animações extensas e efeitos
- **Designs minimalistas/refinados** → contenção, precisão, atenção cuidadosa a espaçamento, tipografia e detalhes sutis

A elegância vem de executar bem a visão — não de adicionar mais elementos.
