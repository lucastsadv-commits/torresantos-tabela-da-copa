# Workflow: High-End Frontend Design

Siga estes passos cronológicos para executar a skill `high-end-frontend` com sucesso:

1. **Definição de Intenção Estética**: Determine o propósito do design. Escolha um "tom" ou "extremo" estético (ex: editorial/revista, industrial/utilitário, minimalista refinado).
2. **Seleção Tipográfica**: Elimine fontes padrão (Inter, Arial, Roboto). Escolha uma fonte Display de alto impacto e uma Serif/Sans de suporte refinada.
3. **Draft da Paleta de Cores**: Evite gradientes genéricos. Defina variáveis CSS focando em cores dominantes arrojadas com acentos intensos, ou um minimalismo monocromático rigoroso.
4. **Geração do Layout Estrutural**: Crie uma composição espacial que quebre expectativas (assimetria, grid quebrado, uso vasto de whitespace).
5. **Fundos e Texturas**: Insira a profundidade com CSS avanzado: grain overlays, noise, meshes, glassmorphism agressivo ou brutalist borders.
6. **Orquestração de Motion**: Implemente entrance animations com delays escalonados (staggered reveals) e micro-interações de hover imprevisíveis, priorizando CSS puro ou framer-motion.
7. **Revisão de Qualidade**: Verifique a regra de complexidade. O código reflete precisamente a visão artística? Se parecer "IA genérica", volte ao passo 4.
