# Premium UI Design

Focado em elevar o valor percebido através do design de interface.

## Princípios de Design

1. **Autoridade**: Uso de cores sóbrias e tipografia impactante (Montserrat/Lato).
2. **Modernidade**: Efeitos de Glassmorphism e gradientes sutis.
3. **Interatividade**: Micro-animações que guiam o usuário (Framer Motion).
4. **Espaçamento**: Uso generoso de margens para evitar poluição visual.

## Identidade Visual (Base)
- **Cores**: Azul Profundo `#0a2f57`, Acento Terracota `#b35120`.
- **Efeitos**: `backdrop-blur-md`, `shadow-xl`.

## Checklist de Qualidade
- O design transmite confiança?
- A hierarquia visual está clara?
- Os CTAs estão destacados sem serem agressivos?
- A performance visual está otimizada?
