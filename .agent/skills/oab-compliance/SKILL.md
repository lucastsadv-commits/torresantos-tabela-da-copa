# OAB Compliance

Esta skill garante a sobriedade e ética exigidas pela OAB em projetos jurídicos.

## Regras de Ouro

1. **Informativo, não imperativo**: Use "Verificar direitos" em vez de "Contrate agora".
2. **Proibição de Mercantilização**: Vedado o uso de "Grátis", "Desconto" ou "Promoção".
3. **Sem Promessa de Resultado**: Nunca use "Garantia de vitória". Use "Análise de viabilidade".
4. **Identificação Obrigatória**: Deve constar nome do advogado e OAB no rodapé.

## Fluxo de Revisão

1. Analisar textos das seções Hero e Footer.
2. Identificar termos sensíveis.
3. Sugerir substituições baseadas no Provimento 205/2021.
4. Validar se o design mantém a sobriedade necessária.
