Este workflow ativa a skill **Creative Brainstorming** para alinhar visões antes da execução.

### Passos:
1. **Ativar Skill**: Carregar as diretrizes de `.agent/skills/creative-brainstorming/SKILL.md`.
2. **Ciclo de Descoberta**: Fazer perguntas iterativas (uma por vez).
3. **Draft de Conceito**: Apresentar blocos de 200-300 palavras para feedback.
4. **Finalização**: Concluir o brainstorming com um resumo claro das decisões.
