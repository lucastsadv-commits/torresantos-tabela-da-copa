Este workflow ativa a skill **Writing Legal Copy** para transformar o wireframe em copy de alta conversão.

### Passos:
1. **Ativar Skill**: Carregar instruções de `.agent/skills/writing-legal-copy/SKILL.md`.
2. **Análise da Estrutura**: Revisar o wireframe gerado pelo `/cro` e a estratégia do `/estrategia`.
3. **Definição de Tom**: Calibrar a voz do advogado (empático para família, firme para criminal, etc.).
4. **Redação do Headline (H1)**: Foco na dor/solução do cliente, sem promessas de resultado.
5. **Corpo do Texto**: Redigir cada seção com linguagem acessível — sem juridiquês sem explicação.
6. **CTAs Éticos**: Escrever botões de ação informativos (ex: "Verificar Meus Direitos").
7. **Checklist OAB**: Verificar que nenhum termo proibido foi utilizado (garantia, grátis, promoção).
8. **Output Markdown**: Entregar o copy completo pronto para ser colado no site.
9. **Transição**: Sugerir `/designer` ou `/front` para o próximo passo de design/código.
