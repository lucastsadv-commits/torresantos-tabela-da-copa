Este workflow ativa a skill **Optimizing Conversion Rates** para estruturar a página com psicologia de conversão.

### Passos:
1. **Ativar Skill**: Carregar instruções de `.agent/skills/optimizing-conversion-rates/SKILL.md`.
2. **Análise Estratégica**: Revisar o output JSON gerado pelo `/estrategia` (persona, dor, abordagem).
3. **Jornada do Usuário**: Mapear o fluxo emocional: Atenção → Interesse (Educação) → Desejo (Autoridade) → Ação.
4. **Estrutura de Seções**: Definir a ordem e propósito de cada seção da LP.
5. **Posicionamento de CTAs**: Indicar onde os botões de ação devem aparecer para máximo efeito.
6. **Output Markdown**: Entregar o "Esqueleto Text Wireframe" da landing page em formato Markdown.
7. **Transição**: Sugerir o próximo comando `/copy` para escrever o texto completo baseado na estrutura.
