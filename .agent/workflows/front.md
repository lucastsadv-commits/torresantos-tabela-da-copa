Este workflow ativa a skill **Coding Frontend Components** para transformar copy e design em código limpo e funcional.

### Passos:
1. **Ativar Skill**: Carregar instruções de `.agent/skills/coding-frontend-components/SKILL.md`.
2. **Revisão dos Insumos**: Verificar que o copy (`/copy`) e as diretrizes visuais (`/designer`) já estão definidos.
3. **Escolha do Framework**: Confirmar stack (React/Vite/Tailwind ou HTML semântico puro).
4. **Estrutura Semântica**: Implementar `<header>`, `<main>`, `<section>`, `<footer>` corretamente.
5. **Botão WhatsApp**: Garantir o formato correto `https://wa.me/55XXXXXXXXXXX?text=...`.
6. **Rastreamento**: Adicionar placeholders para Google Analytics / Meta Pixel nos comentários.
7. **Acessibilidade**: Verificar `aria-label` em todos os botões interativos.
8. **Output**: Entregar o bloco de código pronto (HTML/JSX/TSX) para implementação imediata.
