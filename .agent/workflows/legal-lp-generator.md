# Legal Landing Page Generator

This workflow orchestrates 5 specialized skills to transform a simple "Practice Area + Problem" input into a fully coded, high-conversion landing page.

## Workflow Steps

1.  **Strategy Phase** (CMO)
    -   Skill: `strategizing-legal-marketing`
    -   Action: Analyze the user's input to define the Persona, Pain Point, and Educational Approach.

2.  **Structure Phase** (CRO)
    -   Skill: `optimizing-conversion-rates`
    -   Action: Use the strategy to create a text-based wireframe skeleton focused on conversion.

3.  **Content Phase** (Copywriter)
    -   Skill: `writing-legal-copy`
    -   Action: Write the full, OAB-compliant text content based on the wireframe.

4.  **Design Phase** (Art Director)
    -   Skill: `designing-legal-ui`
    -   Action: Generate a visual style guide (colors, typography, imagery) tailored to the content's tone.

5.  **Implementation Phase** (Developer)
    -   Skill: `coding-frontend-components`
    -   Action: Produce the HTML/CSS code or technical prompt for the final build.

## Usage
Run this workflow when you have a legal niche and a client problem, and need a complete landing page solution.
